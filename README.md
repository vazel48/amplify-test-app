# Amplify Test App

This is a simple static website to test AWS Amplify deployment.

## Structure

amplify-test-app/
├── public/
│   └── images/
│       └── example-image.jpg
├── src/
│   ├── css/
│   │   └── styles.css
│   └── index.html
├── .gitignore
└── README.md


- `index.html`: The main HTML file.
- `css/styles.css`: The CSS file for styling.
- `public/images/`: Contains images used in the site.
